# 🐍 Python Database Integration: PyMySQL & MySQL

This project demonstrates how to connect a Python environment to a MySQL database using the **PyMySQL** library. It covers the full **CRUD** (Create, Read, Update, Delete) lifecycle, specifically designed for use in **Google Colab**.

## 🚀 Key Features
*   **Environment Setup:** Automated installation of MySQL server within a Linux-based cloud environment.
*   **Database Management:** Creating databases and tables dynamically.
*   **CRUD Operations:**
    *   **Create:** Inserting new employee records.
    *   **Read:** Fetching and displaying data with empty-record validation.
    *   **Update:** Modifying existing records (e.g., updating age based on criteria).
    *   **Delete:** Removing records safely using SQL filters.

## 🛠️ Technology Stack
*   **Language:** Python 3.x
*   **Library:** PyMySQL
*   **Database:** MySQL Server
*   **Platform:** Google Colab

## 📖 How to Use
1. Open the `.ipynb` file in Google Colab using the **"Open in Colab"** button.
2. Run the **Environment Setup** cell to initialize the MySQL service.
3. Execute the cells sequentially to perform database operations.
